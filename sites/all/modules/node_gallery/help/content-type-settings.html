While there's not that much that Node Gallery does with content types, there's just a couple tips to give here. First, let's examine the Manage fields tab on the gallery type:



From here, you can drag the Gallery Images view around to reorder where the images are displayed at on your Gallery node page. You want the images above the title? The description above the images? No problem, drag and drop!

Similar to the Gallery content type, the Manage fields tab on the image type also has something of interest:


Here, you can do the same with the image navigator (so long as it's enabled) as you can with the image view above. If you don't like the image navigator at the top, drag it down to the bottom.
