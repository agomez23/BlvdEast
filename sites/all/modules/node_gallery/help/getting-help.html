The Node Gallery community on Drupal.org is an active one. Your best bet is to start in the issue queue.

Often times, themes and other modules can interfere with Node Gallery's operations. You can test a "clean room" implementation of Node Gallery by using the demo site. Visit the project page and click the "Try out a demonstration" link under the "Resources" block on the right.

We've also added a component in the issue queue to point out Howto's, Examples, and FAQ's.
